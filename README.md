# IMDb Movie Review Sentiment Analysis using Python

## Project Description

This project focuses on performing Sentiment Analysis on IMDb movie reviews using Python. The main objective of this project is to analyze textual movie reviews and classify them into Positive, Negative, or Neutral sentiments using Natural Language Processing (NLP) techniques.

The project demonstrates how Python libraries can be used to process text data, perform sentiment classification, and visualize sentiment distribution effectively.

---

## Project Overview

In this project, movie review data from the IMDb dataset is analyzed using TextBlob for sentiment prediction. The reviews are classified based on sentiment polarity, and the results are visualized using graphs.

The workflow includes:
- Loading movie review dataset
- Text preprocessing
- Sentiment classification
- Data visualization
- Insight generation

## Project Tasks

### 1. Data Loading
- Imported IMDb movie review dataset into Google Colab
- Inspected dataset structure and contents

### 2. Sentiment Analysis
- Applied TextBlob sentiment polarity analysis
- Classified reviews into:
  - Positive
  - Negative
  - Neutral

### 3. Data Visualization
- Created bar charts for sentiment distribution
- Visualized review classification results

### 4. Insight Generation
- Identified overall audience sentiment trends
- Analyzed review polarity distribution
